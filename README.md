# AI Travel Itinerary Builder

A modern, AI-powered travel itinerary builder with a beautiful and intuitive user interface. This application helps users create personalized travel plans based on their preferences.

## Features

- Modern and responsive UI design
- Smooth animations and transitions
- User-friendly form for travel preferences
- AI-powered itinerary generation
- Mobile-friendly interface

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
